Terminal Software

A beautiful TUI package manager installer

↑↓ Arrow key navigation for browsing packages
📦 View package details before installing
📊 Progress bar with download speed & ETA
🔧 Supports apt, dnf, pacman, zypper, xbps, apk

Quick Install

curl -sSL https://raw.githubusercontent.com/bobbythedemon/terminal-software/main/install.sh | bash
⬇ Download .deb Package
